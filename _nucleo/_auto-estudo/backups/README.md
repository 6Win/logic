# Backups do auto-estudo

Cópias de segurança de cada arquivo do cérebro ANTES de uma edição de aprendizado
do tipo mutante (ver `_nucleo/auto-estudo.md`). Nome: `AAAA-MM-DD_HHMM_<arquivo>.bak`.

São rede de segurança LOCAL desta máquina — não fazem parte do produto e não vão na
distribuição (o conteúdo `.bak` é gitignored). Restaurar um backup = copiar o `.bak` de
volta por cima do arquivo original e marcar a entrada como revertida no `registro.md`.
