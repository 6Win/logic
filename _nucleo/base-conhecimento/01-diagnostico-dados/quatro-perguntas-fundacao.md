# As 4 perguntas antes de qualquer coisa

Antes de construir oferta, canal ou campanha, responder:

1. **Qual é o problema?** (a dor real que você resolve)
2. **Para quem?** — público-alvo → nicho → **cliente ideal** (gênero, faixa etária, nível de
   experiência, localização, faturamento mensal/anual)
3. **Qual é a solução?** (tecnologia, serviço, processo, conteúdo…)
4. **Como você entrega essa solução?**

Sem essas 4 respondidas, o esforço se apoia numa fundação frágil — "qualquer ventinho derruba".
A maioria que abre negócio e desiste é porque **o esforço não gerou resultado** (fundação torta).

**Aplicação LOGIC:** essas 4 perguntas abrem o **discovery do diagnóstico** — são o esqueleto pra
entender o negócio do cliente antes de diagnosticar. Cruzam direto com os **4 Fits** (Produto ↔
Mercado ↔ Monetização ↔ Canal).
