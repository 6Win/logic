# Fim do perfeccionismo: lança e melhora no percurso

Esperar o "produto perfeito" pra lançar é **autossabotagem**. O perfeito não chega — sempre vai ter
mais uma funcionalidade, mais um detalhe pra polir. Enquanto você aperfeiçoa, **o mercado não para
pra te esperar**. Então: lança, e melhora no percurso, com dado real do mercado.

**Aplicação LOGIC:** casa com dois princípios da Logic —
- O diagnóstico é um projeto de descoberta, não um milagre pronto.
- A primeira versão não precisa estar completa — precisa ser boa o suficiente pra um
  diagnóstico matador. Vende, entrega, e melhora **com** o cliente.
